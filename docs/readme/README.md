# README screenshots

PNG files in this folder are referenced from the root [`README.md`](../../README.md).

**To update the images:** drop new screenshots here using the same filenames:

| File | Typical content |
|------|-----------------|
| `video-tracking-overlay.png` | Match video with court lines + player boxes + ball |
| `live-court-view.png` | 2D bird’s-eye court with A / B / ball positions |
| `serve-placement.png` | Serve landing scatter on court boxes (in vs fault) |
| `player-heatmaps.png` | Side-by-side movement heatmaps for both players |
| `match-stats.png` | Serve / rally / errors / coverage / ball-speed analytics |

After replacing files, commit from the repo root so GitHub shows the new previews.

GitHub renders images with paths like `docs/readme/live-court-view.png` (relative to the repository root).
